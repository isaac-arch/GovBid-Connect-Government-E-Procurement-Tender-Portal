GovBid Connect
Government E-Procurement & Tender Portal

HOW TO RUN

1. Install XAMPP
2. Start Apache
3. Place govbid folder inside:

C:\xampp\htdocs\

4. Ensure data folder exists
5. Ensure tenders.txt, contractors.txt, bids.txt exist
6. Ensure uploads folder exists

7. Open browser:

http://localhost/govbid/

IMPORTANT NOTES

- Apache must be running
- Folder must be directly inside htdocs
- Do not rename php files incorrectly
- File uploads go into uploads folder
- No database required

FEATURES

- Tender publishing
- Contractor registration
- Bid submission with file upload
- Transparency dashboard
- Admin control page
- File-based backend
- Enterprise layout
